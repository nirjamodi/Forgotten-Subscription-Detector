# Forgotten Subscription Detector

## Overview
Forgotten Subscription Detector is a Python-based data analytics project designed to help users identify recurring or forgotten subscriptions from transaction data. By analyzing historical payment records, the system highlights repeated charges that may otherwise go unnoticed, helping users better manage their personal finances.

## Problem Statement
With the growing number of digital subscriptions, users often forget about recurring charges for services they no longer use. These subscriptions continue to deduct money automatically, leading to unnecessary expenses that are difficult to track manually.

## Solution
This project analyzes transaction-level data to:
- Detect recurring payments based on frequency and amount
- Identify potential subscription-based transactions
- Summarize recurring expenses in a clear and user-friendly format

## Key Features
- Upload transaction data in CSV format
- Detect monthly or weekly recurring payments
- Identify merchant names and recurring amounts
- Generate a summary of potential forgotten subscriptions
- Simple and intuitive interface for analysis

## Tech Stack
- **Python**
- **Pandas** for data processing
- **Flask** for web-based interaction
- **CSV** file handling

## Results
- Successfully identifies recurring transaction patterns
- Improves visibility into subscription-based expenses
- Helps users recognize unnecessary or forgotten charges
